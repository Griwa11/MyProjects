# 🏋️ Fitness Trainer Bot

Телеграм-бот помощник для фитнес тренера.

## 📋 Возможности

### Для клиентов:
- ✅ Заполнение анкеты
- 📅 Просмотр тренировок
- 📊 Отслеживание прогресса

### Для тренера:
- 👥 Управление клиентами
- 📅 Назначение тренировок
- 💳 Отслеживание оплаченных тренировок
